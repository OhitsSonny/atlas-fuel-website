# Atlas Fuel Australia — Website

A multi-page corporate website for **Atlas Fuel Australia Pty Ltd**, the Kwinana-based bulk fuel and logistics partner serving mining, marine, agriculture, transport and retail across Australia.

## Live preview

Run a local server from the project root:

```bash
node serve.mjs
```

Then open [http://localhost:3000](http://localhost:3000).

## Site map

- **/** — Homepage (hero, services grid, trust panel, leadership, sustainability, careers, news)
- **/about.html** — About Atlas Fuel, story, vision & purpose, core values, timeline
- **/what-we-do.html** — All 11 service lines (hub page with anchored sections)

## Tech

- Plain HTML + a single shared `styles.css`
- Custom CSS design system (no Tailwind, no build step)
- Google Fonts: **Manrope** (display) + **Inter** (body)
- Vanilla JS for sticky nav, mobile drawer, scroll-reveal
- Cinematic CSS image enhancement (filter + layered gradient overlays)
